## Ejecutar en desarrollo

1. Clonar del repositorio
2. Ejecutar

```bash
$ yarn install
```

3. Tener Nest CLI instaladao

```
$ npm i -g @nestjs/cli
```

4. Levantar la base dedatos

```
$ docker-compose up -d
```
